# MaiLee Test

Simple Test with mocha and chai

## Requirements

- [Node.js](https://nodejs.org/en/)
- Optional: [Yarn](https://yarnpkg.com/en/docs/install)

## Usage

### Using NPM

- `npm install`
- `npm test`
- `npm run lint`

### Using Yarn

- `yarn install`
- `yarn test`
- `yarn lint`
